server side first (this repo)  
clone this and cd into that directory  
`bundle install`  
`rake db:migrate`  
`rake db:seed`  
`bundle exec rails s -p 3001`  

now client  
go to https://github.com/jd2rogers2/rr-ecommerce-client  
clone that and cd into it  
`npm install`  
`npm start`
